# Screenshot Capture Guide — ITDR

---

### 01 — Sentinel Analytics Rules: ITDR Active
**Path:** Sentinel > Analytics > Active rules
**Show:** All ITDR rules listed with status, severity, last run

### 02 — Entra ID Protection Risk Policy
**Path:** Entra ID > Security > Identity Protection > Policies
**Show:** User risk policy + sign-in risk policy configuration

### 03 — Incident Timeline: Tabletop Exercise
**Path:** Sentinel > Incidents > [tabletop incident] > Timeline
**Show:** Full timeline from first signal to containment with timestamps

### 04 — Playbook Execution Evidence
**Path:** Sentinel > Automation > Playbook runs
**Show:** playbook-revoke-contain execution: all 5 steps completed with timestamps

### 05 — MTTD/MTTR Dashboard
**Path:** Sentinel > Workbooks > ITDR Metrics (custom)
**Show:** Monthly MTTD/MTTR averages, incident count by tier, false positive rate
